# Workflow Diagram (Text Representation)

Lead Source → Capture → CRM Storage → Qualification → AI Response → Follow-Up

## Flow

1. Lead enters via form or DM
2. Data stored in Notion/Airtable
3. Classified (high/medium/low intent)
4. AI generates response
5. Message sent
6. Follow-up triggered if no reply

## Insight
This replaces manual outreach with an automated pipeline.
